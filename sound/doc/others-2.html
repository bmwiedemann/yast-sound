2. Sound card drivers

Sound card drivers are (usually) kernel modules (for kernel >= 2.2.x). It means they have to be loaded at some time. This can be done by hand (with /sbin/insmode) or during booting (with /etc/conf.modules).

2.1 Alsa-drivers

These drivers are compatible with the OSS/Free and OSS/Linux sound drivers (the drivers in the kernel), but has its own interface that is even better than the OSS drivers. A list of features can be found at http://www.alsa-project.org/intro.html.

The list of ALSA Supported sound cards can be found at http://www.alsa-project.org/src/soundcards.html.

The list of alsa-driver modules can be found at apendix.

requires: kernels >= 2.2.x

provides: sound card drivers (as kernel modules)

2.2 OSS/Free-drivers

Drivers from Open Sound System are now part of Linux kernel.

The list of OSS/Free supported sound cards, which are not supported by Alsa can be found in apendix.

requires: nothing

provides: sound card drivers (as kernel modules)
