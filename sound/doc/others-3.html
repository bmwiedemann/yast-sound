3. Device files

For proper work, devices file entries must be created for sound devices. These are normally created during instalation. Alsa drivers uses their own devices (files in /dev directory).

Alsa package contains shell script (snddevices) for creating them. If the kernel has DEVFS support the script needs not to be run.

requires: snddevices or DEVFS kernel support

provides: alsa sound devices
